Сборка приложения

Требования
Должен быть установлен Java Development Kit (JDK) версии 8 или выше.

Шаги сборки
- Откройте терминал (командную строку).
- Используйте команду Git для клонирования репозитория:
  https://github.com/LuckyAm20/Java_sem5.git
- Перейдите в каталог проекта:
  cd "Java project 5 sem"/src
- Используйте команду Javac для компиляции Java-кода:
  javac Huffman.java
